Welcome

The Angular coding exercise is a way for us to learn about your coding style.

Tasks

  1. Wire up the routing framework of your choice
  2. Build a screen where you can list out existing users
  3. Add a screen for adding a new user
  4. Add a screen for editing a user or reuse the new screen
  5. Add a function for deleting a user from the users list
  6. Make the list of users sortable by name or by email address
  7. Make the list of users filterable by name or by email address or both

Endpoints

  • Get all users from /api/users
  • Create a new user by posting to /api/users
  • Get user by getting /api/users/:userId
  • Delete user by deleting /api/users/:userId
  • Update user by putting /api/users/:userId
